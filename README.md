## Quick start

```
import Treemap from 'treemap-chart';
```
or
```
const Treemap = require('treemap-chart');
```
or even
```
<script src="//unpkg.com/treemap-chart"></script>
```
then
```
var myGraph = Treemap();
myGraph(<myDOMElement>)
    .prop(...);
```
